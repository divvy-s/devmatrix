# CastKit Global Feed 🚀

A complete Farcaster-inspired feed page built entirely in Next.js 14, Tailwind, and React.

## What's included
- Fully responsive dark-mode UI exclusively using Tailwind CSS.
- Complete type safety via TypeScript.
- Optimistic updates for Likes and Comments functionality.
- Debounced component-level User Searching.
- Extensible Web3 Tippping and ML Ranking architecture.
- Full server+client hydration with loading skeletons.

## How to run
1. Ensure your Next.js application is running (`npm run dev`)
2. Navigate to `/feed` to see the global feed with mock data.
